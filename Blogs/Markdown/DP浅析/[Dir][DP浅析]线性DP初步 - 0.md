# #1.0 线性DP的基本知识

## #1.1 啥是线性DP

首先，线性DP**不代表**线性时间复杂度，它的范围很广，如果一个动态规划算法的 “状态” 包含多个维度，但在每个维度上都具有 “线性” 变化的 “阶段”，也可以称为**线性动态规划**

## #1.2 三要素 & 三条件

### #1.2.1 三要素

- 状态
- 阶段
- 决策

### #1.2.2 三个基本条件

- 子问题重叠性
- 无后效性
- 最优子结构性质

以上三要素及三条件适用于所有种类的DP
